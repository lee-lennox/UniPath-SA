# UniPath SA

AI-powered platform helping South African students calculate APS, explore careers, and find university programmes.

---

## About the Project

UniPath SA is a full-stack education platform designed to simplify university applications in South Africa by helping students:

- Calculate APS scores based on university rules  
- Discover suitable careers based on interests  
- Find university programmes across institutions  
- Check eligibility for admission  
- Understand academic pathways more clearly  

---

## Problem It Solves

Students often struggle with:

- Different APS calculation methods per university  
- Scattered information across multiple websites  
- Unclear career-to-degree pathways  
- Difficulty knowing which programmes they qualify for  

This platform centralizes all of that into one system.

---

## Features

- APS Calculator (University-specific rules)
- Career Recommendation System
- Programme Finder
- Eligibility Checker
- University Explorer
- Prospectus Access (planned)
- Application Tracking (planned)
